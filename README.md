# Gushwork Web Page Assignment

A high-fidelity, responsive web page built with vanilla HTML, CSS, and JavaScript. This project showcases a premium frontend implementation following strict design specifications.

## Live Demo
[View Live Site](https://gushwork-assignment-pi.vercel.app/#)

## Repository
[GitHub Repository](https://github.com/LoveshPurswani18/gushwork-assignment)

## Key Features
- **Pixel-Perfect Implementation**: Meticulously followed Figma design specs for desktop, tablet, and mobile.
- **Interactive Hero Zoom**: Real-time magnifying zoom with dynamic ratio calculation for product inspection.
- **Responsive Navigation**: Fluid header that adapts across 6 different breakpoints.
- **Sticky Mini-Header**: Intelligent secondary header that appears after the first fold for continuous CTA access.
- **Manufacturing Stepper**: Synchronized tab system with smooth content transitions and mobile-optimized badges.
- **Multi-Modal Flow**: High-fidelity glassmorphism modals for "Request a Quote" and "Technical Datasheet" downloads.
- **FAQ Accordion**: Clean, interactive FAQ section for user guidance.

## Technology Stack
- **HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS3**: Modern layout techniques using Flexbox and CSS Grid.
- **Vanilla JavaScript**: Pure ES6 logic for interactions without any frameworks or libraries.

## Project Structure
- `index.html`: Main structural markup with optimized image loading.
- `styles.css`: Comprehensive design system with responsive variables and animations.
- `script.js`: Interactive logic for zoom, sliders, tabs, and modal management.
- `assets/`: Optimized SVG and raster images for the portfolio and UI.

## Performance & Accessibility
- **Performance**: Asynchronous image decoding and lazy loading implemented for all below-the-fold assets.
- **A11y**: ARIA labels provided for all interactive icon buttons; logical heading hierarchy maintained.
- **Cross-Browser**: Compatible with modern Chrome, Safari, and Firefox.
